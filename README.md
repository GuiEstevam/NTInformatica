# NT Informática - Site Corporativo

Site corporativo estático para apresentação de serviços de suporte técnico e soluções em tecnologia da informação.

## Stack Tecnológica

- **Frontend**: HTML5, CSS3 (SASS), JavaScript (ES6+)
- **Build Tool**: Vite 5
- **Ícones**: Font Awesome + Ionicons
- **Template Base**: HTML5 UP Editorial (adaptado)

## Instalação

```bash
npm install
```

## Desenvolvimento

```bash
npm run dev
```

O site estará disponível em `http://localhost:3000`

## Build para Produção

```bash
npm run build
```

Os arquivos compilados estarão em `dist/`

## Estrutura do Projeto

```
nt-informatica/
├── index.html
├── assets/
│   ├── css/          # Estilos CSS
│   ├── js/            # JavaScript
│   ├── sass/          # Fonte SASS
│   └── images/        # Imagens
└── dist/              # Build de produção
```

## Características

- Design minimalista com identidade rockeira sutil
- Paleta monocromática (preto/branco)
- Tipografia geométrica moderna
- Totalmente responsivo
- Suporte a dark mode
- Sistema de toast global
- Formulário de contato integrado
- SEO otimizado
